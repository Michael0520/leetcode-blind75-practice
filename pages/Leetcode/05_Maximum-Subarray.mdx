# LeetCode 53: Maximum Subarray

> 陣列、連續子陣列和、Kadane 算法

## 題目描述

給定一個整數陣列 `nums`，目標是找出具有最大和的連續子陣列（subarray）並返回其和。

### 限制條件

- 陣列長度在 \(1\) 到 \(10^5\) 之間。
- 整數範圍為 \(-10^4\) 刀 \(10^4\)。

### 範例

例如，對於 `[-2, 1, -3, 4, -1, 2, 1, -5, 4]`：

- 連續子陣列 `[4, -1, 2, 1]` 具有最大和 6。
- 所以，返回的答案就是 6。

## 解題思路

### 方法一：Kadane 算法（時間複雜度 \(O(n)\)）

1. **局部最大和（Current Max）**：儲存到目前為止的最大和，初始化為 `nums[0]`。
2. **全局最大和（Global Max）**：儲存全局的最大和，初始化為 `nums[0]`。

#### Code

```javascript
function maxSubArray(nums) {
  let currentMax = nums[0];
  let globalMax = nums[0];

  for (let i = 1; i < nums.length; i++) {
    currentMax = Math.max(nums[i], currentMax + nums[i]);
    globalMax = Math.max(globalMax, currentMax);
  }

  return globalMax;
}
```

## 測試

```javascript
Copy code
console.log(maxSubArray([-2, 1, -3, 4, -1, 2, 1, -5, 4])); // 應返回 6
console.log(maxSubArray([1])); // 應返回 1
console.log(maxSubArray([5, 4, -1, 7, 8])); // 應返回 23
```
