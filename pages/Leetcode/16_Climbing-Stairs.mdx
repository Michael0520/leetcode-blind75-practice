# LeetCode 70: Climbing Stairs

> 動態規劃

## 題目描述

有一個含有 `n` 階的樓梯，你的目標是從底部爬到頂部。每次爬梯子時，你可以選擇爬 1 階或 2 階。

問題是：有多少種獨特的方式可以爬到樓頂？

### 限制條件

- `1 <= n <= 45`

## 解題思路

### 動態規劃（Dynamic Programming）

用動態規劃來解這個問題，我們用一個數組 `dp` 來存儲到達每一階的方法數。

#### 解法步驟

自我理解：
由於 0 & 1 階梯時，都只有個算法，所以我們需要專注在 2 階梯以上（包含 2 階梯）的情況。

- 到達第 1 階的方法數 = 1
- 到達第 2 階的方法數 = 2
- 到達第 3 階的方法數 = 到達第 1 階的方法數 + 到達第 2 階的方法數
- 到達第 4 階的方法數 = 到達第 2 階的方法數 + 到達第 3 階的方法數
  以此類推...

1. **初始化數組**
   - 初始化一個長度為 `n+1` 的數組 `dp`，其中 `dp[0] = 1` 和 `dp[1] = 1`。
2. **填充數組**
   - 從 `i = 2` 開始到 `n`，計算 `dp[i] = dp[i-1] + dp[i-2]`。

---

## Code

```javascript
const climbStairs = (n) => {
  let dp = new Array(n + 1).fill(0);
  dp[0] = 1;
  dp[1] = 1;
  for (let i = 2; i <= n; i++) {
    dp[i] = dp[i - 1] + dp[i - 2];
  }
  return dp[n];
};
```

## 測試與調試

```javascript
console.log(climbStairs(2)); // 應返回 2
console.log(climbStairs(3)); // 應返回 3
console.log(climbStairs(4)); // 應返回 5
```
